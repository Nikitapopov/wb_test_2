Что выведет программа? Объяснить вывод программы.

```go
package main

func main() {
	ch := make(chan int)
	go func() {
		for i := 0; i < 10; i++ {
			ch <- i
		}
	}()

	for n := range ch {
		println(n)
	}
}
```

Ответ:
```
Программа выведет числа от 0 до 10 не включительно и зависнет из-за дедлока.

Объявляем канал, запускаем горутину.
Далее попадаем в цикл итерации чтения канала. Когда горутина записывает очередное значение от 0 до 10 по порядку, происходит чтение в цикле и вывод числа.
После того, как горутина заканчивает работу, цикл чтения канала остается работать.

Как исправить: закрыть канал после итерационной записи.
```
